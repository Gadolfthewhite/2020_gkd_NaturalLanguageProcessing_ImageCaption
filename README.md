# 2020_gkd_NaturalLanguageProcessing_ImageCaption
2020年秋季国科大自然语言处理（刘洋）课后编程作业，任务为图像标注。

需要的包主要有：
1. Django 3.1.4
2. PyTorch 1.2.0
3. torchvision 0.4.0
4. scipy 1.2.0
5. PIL 7.2.0
6. tqdm 4.49.0
7. skimage 0.17.2
8. numpy 1.15.4

首先需要将上述python包装上。

然后需要将模型、语料库文件和虚拟环境下载到根目录下（venv压缩文件需要解压），百度链接如下：
链接：https://pan.baidu.com/s/1Hbh62e-z3PoefckiM5t8rA 
提取码：3atn 

最后，需要在PyCharm中加载虚拟环境即可。
